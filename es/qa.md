# Aseguramiento de la Calidad (QA)

Este documento lista herramientas estándar para la revisión de calidad de código y mejora continua.

## Análisis estático

- SonarQube (`https://www.sonarsource.com/products/sonarqube/`): plataforma de análisis estático, detección de code smells, vulnerabilidades y métricas de mantenibilidad.

## Revisión de código asistida por IA

- CodeRabbit (`https://www.coderabbit.ai/`): asistente de revisión de código con IA integrado a pull requests para sugerencias de estilo, corrección y buenas prácticas.

## Lineamientos

- Integrar verificaciones en CI para cada pull request.
- Bloquear merges ante issues críticos; gestionar deuda técnica con responsables claros.
- Documentar falsos positivos y aplicar supresiones específicas en lugar de ignores globales.
